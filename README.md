# testurtk1
Разработать консольное приложение "Калькулятор", выполняющее базовые арифметические операции над двумя числами. Приложение должно обрабатывать введённые пользователем данные и выводить результаты вычисление. Приглашения к вводу должны быть понятными для пользователя. Каждая операция должна быть в отдельном методе.


Алгоритм работы:
1. Вывести инструкцию
2. Получить ввод от пользователя
3. Проверка данных
4. Вывполнение операций
5. Вывод результата

   
Операции:
1. Сложение (+)
2. Вычитание (-)
3. Умножение (*)
4. Деление (/)


Тест-кейсы:

Сложение

1. Ввод: 5 + 3
   Ожидаемый результат: 8
2. ВычитаниеВвод: -5 + -3
   Ожидаемый результат: -8
   
Вычитание

1. Ввод: 10 - 4
   Ожидаемый результат: 6
2. Ввод: 4 - 10
   Ожидаемый результат: -5
   
Умножение

1. Ввод: 4 * 5
   Ожидаемый результат: 20
2. Ввод: -5 * 4
   Ожидаемый результат: -20
   
Деление

1. Ввод: 10 / 2
   Ожидаемый результат: 5
2. Ввод: 5 / 2
   Ожидаемый результат: 2,5
