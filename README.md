# testurtk1
Разработать консольное приложение "Калькулятор", выполняющее базовые арифметические операции над двумя числами. Приложение должно обрабатывать введённые пользователем данные и выводить результаты вычисление. Приглашения к вводу должны быть понятными для пользователя.
Операции:
1. Сложение (+)
2. Вычитание (-)
3. Умножение (*)
4. Деление (/)
